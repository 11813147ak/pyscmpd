Introduction
------------

Python based sound-cloud music server talking MPD protocol.

Resource Core
-------------

Everything related to songs and managing songs is called a resource. The following resources are known:

                                     +----------+
                                     | Resource |
                                     +----------+
                                         /|\
                                          |
                                     +----+-----+
                                     |          |
                               +-----------+ +------+
                               | Directory | | File |
                               +-----------+ +------+
                                    /|\        
                                     | 
                               +----------+
                               | Provider |
                               +----------+
