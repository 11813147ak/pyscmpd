Prerequisites
-------------

* [python soundcloud API installed] (https://github.com/soundcloud/soundcloud-python) 
* [python-mpd-server API installed] (http://pympdserver.tuxfamily.org/)
* python-gst installed (apt-get install python-gst0.10)
